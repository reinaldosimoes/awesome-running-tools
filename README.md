### Adding New Tools Format

Please use the following template when adding a new tool:

```markdown
[Tool Name](full URL) ‚Äî Brief description of what it does (1-2 sentences).
```

#### Example:

```markdown
[WeatherToRun](https://weathertorun.app/) ‚Äî Weather-based running condition advisor that helps determine optimal running conditions.
```

#### Quick Checklist

- [ ] The tool is running-related and useful for runners
- [ ] The link works and the tool is accessible
- [ ] The description is clear and concise
- [ ] The tool fits the most appropriate category
